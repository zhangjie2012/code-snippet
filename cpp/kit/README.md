# kit

## 简介

kit 是一个 Linux 下 C/C++ 基础组件库。

适配平台: Linux, Mac

生成: `./autogen.sh && make`

清理: `./autoclean.sh`

## 组件

+ Any
+ Assert
+ CompositeKey, EntryIndex, EntryManager
+ DateTime
+ Delegate
+ FileChannel, Logger
+ MMapData, MMapVector
+ NetAddr
+ NonCopyable
+ ObjectFactory
+ ObjectPool
+ Random, GuidGen
+ SignalHook
+ Singleton
+ Thread
